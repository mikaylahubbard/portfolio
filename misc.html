1. Simple Data Analytics Dashboard

Summer 2025

This project was based off of an internship interview prompt. The goal was to organize and display CSV weather data. The majority of time dedicated to this project was spent within Script.js: sorting the data into a format I could use to display on the dashboard and creating a clean way to display each day responsively. I utililized the Bootstrap CSS framework and the library Chart.js in order to implement the visuals.

Github: https://github.com/mikaylahubbard/InternAssessment
